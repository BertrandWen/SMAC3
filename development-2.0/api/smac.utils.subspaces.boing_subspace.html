class BOinGSubspace(LocalSubspace):

“”” Subspace for BOinG optimizer. Each time we create a new epm model for the subspace and optimize to maximize the acquisition function inside this subregion.

acq_optimizer_local: Optional[AcquisitionFunctionMaximizer]

Subspace optimizer, used to give a set of suggested points. Unlike the optimizer implemented in epm_chooser, this optimizer does not require runhistory objects.
